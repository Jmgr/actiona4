# Enumeration: DayOfWeek


Day of the week, used with `datetime.waitForSchedule`.

```ts
// Wait until next Monday at 09:00
await datetime.waitForSchedule({ dayOfWeek: DayOfWeek.Monday, hour: 9 });
```
