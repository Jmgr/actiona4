# Type Alias: DurationLike

> **DurationLike** = [`number`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Number) \| [`string`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/String)

A duration as milliseconds (number) or a humantime string.

```ts
sleep(500);     // 500 milliseconds
sleep("1s");    // 1 second
sleep("500ms"); // 500 milliseconds
```
