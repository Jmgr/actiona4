# Introduction

Actiona 4 is a cross-platform desktop automation runtime with a TypeScript and JavaScript API.

## What Is Here

- API reference generated from the runtime declarations
- Core classes and types for scripts
- Runtime objects for mouse, keyboard, clipboard, windows, audio, and more

## Running Scripts

Use the CLI during development:

```sh
actiona-run <script.ts>
actiona-run eval "println('hello')"
actiona-run repl
```

Continue with the API section in the sidebar to explore available classes, functions, and variables.
